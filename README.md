## 📌 Project Overview

AI-Powered Customer Support & Service Automation is a Salesforce-based application that automates customer support operations using Salesforce Flow, Prompt Builder, Agentforce, Validation Rules, Reports, Dashboards, and Lightning App Builder.

The system reduces manual work by automatically managing customer cases, validating records, routing cases, and generating AI-powered summaries.

---

## 🎯 Objectives

- Automate customer support operations
- Reduce manual documentation
- Improve case management
- Generate AI-based summaries
- Demonstrate Salesforce Agentforce implementation
